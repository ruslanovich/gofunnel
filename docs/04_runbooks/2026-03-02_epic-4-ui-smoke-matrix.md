# Epic 4 Smoke Matrix (UI closeout)

Короткий финальный smoke checklist для PR-4.6 и закрытия Epic 4.

## Preconditions

- Запущен HTTP server (`npm run http:start:local` или эквивалент).
- Есть тестовые пользователи: обычный пользователь и администратор.
- Для `/share/:token` есть валидный share token.

## Matrix

- `/login`
  - Успешный вход переводит в `/app`.
  - Неверные данные показывают ошибку на русском.
  - Сетевой/общий сбой показывает `Не удалось ...` и кнопку `Повторить`.

- `/request-access`
  - Успешная отправка показывает success-state и ссылку на `/login`.
  - Валидация (`400`) подсвечивает форму и показывает RU-ошибку.
  - Лимит (`429`) показывает RU-ошибку без раскрытия внутренних деталей.

- `/app`
  - Пустой список: корректный empty state.
  - Непустой список: строки, статусы, badges и локальные controls отображаются корректно.
  - Ошибка загрузки списка показывает retry-flow (`Повторить`).

- Overlay (детали файла)
  - Все статусы на русском: processing/success/failed/not found.
  - Закрытие работает всеми путями: крестик, кнопка в footer, backdrop, `Esc`.
  - В состоянии `failed` код/сообщение ошибки выводятся безопасно как текст.

- `/admin/access-requests`
  - Список отображается; локальные поиск/фильтр работают.
  - Actions (`Одобрить`, `Отклонить`, `Связались`, `Создать приглашение`) работают без регрессий.
  - Invite link можно скопировать, статусы обновляются в таблице.

- `/admin/users`
  - Список отображается; локальные фильтры работают.
  - `Отключить`/`Включить` обновляют статус пользователя без регрессий.

- Placeholders (`/share/:token`, fallback `/app/*`, `/admin`)
  - Единый стиль placeholder-блока.
  - Только RU-copy.
  - Есть действие возврата в `/app`.

## Expected closeout result

- В scope Epic 4 нет английских user-facing строк.
- Компоненты и состояния визуально консистентны.
- Smoke-пункты выше выполняются без регрессий API/роутов.
